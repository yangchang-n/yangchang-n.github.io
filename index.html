Unity 2D C# Tilemap 1인 개발 2024.03 – 2024.06

Unity 2D 엔진으로 제작한 횡스크롤 액션 플랫포머입니다. 플레이어 캐릭터의 이동·점프·대시·콤보 공격 시스템을 State Machine 패턴으로 구현하여 확장성과 가독성을 높였습니다. 총 10개의 스테이지와 2종의 보스를 제작하였으며, 씬 전환 및 세이브 시스템도 직접 구현하였습니다.

주요 구현 사항

  • Finite State Machine 기반 플레이어 컨트롤러 설계 및 구현
  • Scriptable Object를 활용한 적 AI 행동 데이터 분리 관리
  • Object Pooling으로 파티클·투사체 성능 최적화
  • PlayerPrefs + JSON 조합의 세이브·로드 시스템 구현
